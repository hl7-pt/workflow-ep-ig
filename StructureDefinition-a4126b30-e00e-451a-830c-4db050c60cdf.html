Extension: Nationality

Official URL: https://hl7.pt/fhir/StructureDefinition/Nationality Version: 1.0.0
Draft as of 2021-10-28 Computable Name: Nationality

Extensão utilizada para registar a nacionalidade do utente, alinhando-se com os padrões administrativos portugueses. Essencial para alguns tipos de estudos ou padrões.

Context of Use

This extension may be used on the following element(s):

  • Element ID Patient
  • Element ID Practitioner

Usage info

Usage:

  • Use this Extension: PT_Patient and PT_Practitioner
